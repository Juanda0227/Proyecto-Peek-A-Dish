# Peek a Dish

Proyecto web que permite visualizar platos de un restaurante antes de visitarlo.

## Funcionalidades
- Visualización dinámica desde JSON
- Sistema de favoritos con localStorage
- Interfaz responsive

## Tecnologías
- HTML
- CSS
- JavaScript
